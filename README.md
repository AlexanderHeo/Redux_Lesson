# Redux Lesson
A small lesson in how to use Redux

## Getting Started
1 clone the repo to your local machine
```
git clone https://github.com/AlexanderHeo/Redux_Lesson.git redux-lesson
```
2 change directory
```
cd redux-lesson
```
3 start project with npm
```
npm run dev
```
